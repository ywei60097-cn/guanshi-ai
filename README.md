# 观时 · 日常命理助手（原型）

这是一个不依赖框架的移动端网页原型，承载了 `PLAN.md` 和 `SPEC.md` 中 v1 的主要体验：

- 出生资料建档（仅存浏览器本地）；
- 首页“今日”建议；
- 穿衣、出行、工作社交、选时间、麻将座位五类场景；
- 可展开的推演依据；
- 赛博命理仪表盘视觉：深色网格、单一动态信号色、细标题、无嵌套卡片。

## 本地运行

在项目根目录执行：

```bash
python3 -m http.server 4173
```

打开 `http://127.0.0.1:4173`。

## 当前命理引擎状态

排盘与规则被拆为两层：

- `bazi-engine.js` 使用 `lunar-javascript` 计算按节气换月的四柱、藏干、十神与流日；
- 规则顺序固定为：月令与格局候选 → 调候 → 旺衰辅助 → 流通与作用 → 场景映射；
- 子时换日是显式用户配置，支持零点换日与子初换日；
- 将结论映射为颜色、方位与行动节奏；
- 每次输出保存规则 ID、来源锚点、事实、推论与限制；
- 明确标注不确定性，不生成输赢、疾病、投资等确定性预测。

运行校验：

```bash
npm run check
```

它仍不是可直接上线的专业命理系统。真太阳时、完整格局成败、相神、从格和大运流年仍需继续实现，并须经过命理顾问的例盘审核。详见 `RULEBOOK.md`。
