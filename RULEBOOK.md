# 观时规则书｜子平推演内核 v0.2

## 目的

本规则书把传统命理的判断顺序变成可执行、可测试、可追溯的产品规则。它不把《子平真诠》《滴天髓》当作可任意拼接的文案库，也不让语言模型自行选择用神。

本文件记录的是对古籍章节的产品化概括，不长篇复制原文；每条线上规则应由命理顾问以其使用版本复核。

## 两部书的规则锚点

| 锚点 | 章节 | 产品化原则 |
| --- | --- | --- |
| `ZPZQ.YONGSHEN` | 《子平真诠》“论用神”“论用神变化”“论相神紧要” | 先取月令与格局条件，再讨论用神、相神及成败。 |
| `ZPZQ.CLIMATE` | 《子平真诠》“论用神配气候得失” | 寒暖燥湿必须进入判断；扶抑不能覆盖调候。 |
| `DTST.MONTH` | 《滴天髓》“月令”“衰旺”“中和” | 以月令与全局旺衰审察体用，不把单一五行数量当结论。 |
| `DTST.FLOW` | 《滴天髓》“源流”“通关”“配合” | 合冲生克须看是否流通、是否具备条件；不能离开全局单断。 |

## 强制顺序

```text
按节气排四柱
  → 月令司令藏干与十神（格局候选）
  → 透干、通根、破格、相神（规则覆盖中）
  → 寒暖燥湿的调候观察
  → 旺衰与扶抑（辅助层）
  → 大运 / 流年 / 流月 / 流日 / 流时的作用
  → 场景映射：颜色、方位、节奏、沟通
```

低优先级步骤不能推翻高优先级步骤。例如“某元素较少”不能跳过月令和格局，直接推出“应补某元素”。

## v0.2 已覆盖

- 通过 `lunar-javascript@1.7.7` 按节气计算四柱。
- 支持显式切换零点换日 / 子初换日，避免隐藏流派假设。
- 输出藏干、十神、地势、月令司令藏干和格局候选。
- 记录调候观察项、旺衰初判与原局—流日的支合冲信号。
- 每条结果包含 `事实 → 推论 → 限制`，界面可查看规则 ID。

## v0.2 明确不做

- 不将“格局候选”伪称为已成立的格局。
- 不在未覆盖透干、通根、破格、相神、从格条件前，宣称得到完整用神。
- 不用神煞、纳音或五行缺失主导结果。
- 不对健康、法律、投资、婚姻结果或博彩输赢作确定判断。

## 下一批必须审校的规则

1. 格局候选的透干、通根、成格与破格条件。
2. 《子平真诠》用神变化、相神配合的可执行规则表。
3. 《滴天髓》通关、源流、顺逆、寒暖燥湿的冲突处理。
4. 真太阳时、出生地经纬度与历史时区校正。
5. 大运起运、流年流月流时及场景映射的例盘回归测试。

## 审核门槛

任何新增规则至少要附带：规则 ID、适用条件、排除条件、来源章节、通俗解释、匿名例盘、自动化测试与命理顾问审核状态。规则有分歧时，产品必须暴露配置或显示“不确定”，而不是偷偷选择一个说法。
