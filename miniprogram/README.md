# 原生微信小程序工程

这是可直接由微信开发者工具导入的原生小程序，不依赖 `web-view`。所有页面都调用同一个服务端规则 API，因此网页原型和小程序不会各自演化出不同的命理结论。

## 使用方法

1. 在 `utils/config.js` 将 `API_BASE_URL` 改为火山引擎部署后的 HTTPS 域名。
2. 在微信小程序后台将该域名配置为服务器域名。
3. 用微信开发者工具导入项目根目录，填写真实 AppID，并在真机预览中完成全流程测试。
4. 在发布前填写隐私保护指引：出生日期、出生时间、出生地的计算目的、存储方式、保存期限、删除入口与联系方式。

## 当前工作模式

- 小程序本地只保存用户主动填写的资料和最近一次结果。
- API 是无状态计算服务：不保存或记录请求体，因此首发不需要数据库。
- 跨设备命盘、历史记录、订阅和账户体系必须等微信登录、加密存储、删除机制完成后再开放。

## 所有保留模块

- 建档和子时换日选择；
- 今日命盘与三条日常建议；
- 穿衣、出行、工作推进、选时间、座位娱乐参考；
- 场景结果与可追溯推演依据；
- 本机资料删除。
